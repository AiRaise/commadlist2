# CCAGI 起動ガイド（初心者向け）

はじめてClaude Codeを使う方のためのステップバイステップガイドです。

---

## Step 1: VS Codeを起動する

VS Codeのアイコンをダブルクリックして起動します。

```
見つからない場合 → Spotlight検索（Cmd + Space）で「Visual Studio Code」と入力
```

---

## Step 2: フォルダを開く

画面左上の **紙のマーク（エクスプローラー）** をクリックして、
**「フォルダーを開く」** ボタンを押します。

```
場所: サイドバー左上 → 📄 アイコン → 「フォルダーを開く」
```

---

## Step 3: 作業フォルダを選択する

作業用のフォルダを選びます。
新しく始める場合は、**新しいフォルダを作成**してください。

```
⚠️ フォルダ名は英語で付けてください（日本語NG）

OK → my-project, todo-app, website
NG → マイプロジェクト, 新規案件, テスト
```

---

## Step 4: ターミナルで CCAGI を初期化する

VS Code内でターミナルを開き、以下のコマンドを入力します。

```
ターミナルの開き方: メニュー「ターミナル」→「新しいターミナル」
             または Ctrl + `（バッククォート）
```

ターミナルが開いたら入力：

```
ccagi-sdk init
```

→ CCAGI環境がセットアップされます（初回のみ）

---

## Step 5: Claude Code を起動する

初期化が完了したら、同じターミナルに以下を入力します：

```
claude
```

→ Claude Codeが起動し、AIと対話できるようになります。

---

## まとめ

```
VS Code起動 → フォルダを開く → ccagi-sdk init → claude
```

以上で準備完了です。あとはClaudeに話しかけるだけ！
